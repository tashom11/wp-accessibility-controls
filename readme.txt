Permet aux visiteurs de personnaliser les paramètres d'accessibilité (taille du texte, police, contraste, espacement) pour améliorer leur expérience de navigation.

== Description ==

WP Accessibility Controls est un plugin WordPress qui ajoute un panneau de contrôle d'accessibilité directement sur votre site. Les visiteurs peuvent facilement :

* **Taille du texte** : Ajuster la taille de la police (5 niveaux)
* **Police** : Choisir parmi différentes polices (y compris OpenDyslexic pour la dyslexie)
* **Hauteur de ligne** : Modifier l'espacement entre les lignes
* **Espacement des lettres** : Ajuster l'espacement entre les caractères
* **Contraste** : Changer le contraste des couleurs (y compris mode inversé)
* **Alignement du texte** : Personnaliser l'alignement du texte
* **Curseur** : Agrandir la taille du curseur si nécessaire

Les paramètres sont sauvegardés et persistent d'une visite à l'autre, que l'utilisateur soit connecté ou non.

== Caractéristiques ==

* Interface simple et intuitive
* Bouton flottant discret mais accessible
* Paramètres persistants via cookies ou métadonnées utilisateur
* Design responsive pour mobile et desktop
* Respect des standards WCAG
* Support du mode sombre
* Aucune configuration requise

== Installation ==

1. Téléchargez le plugin
2. Uploadez le dossier `wp-accessibility-controls` dans `/wp-content/plugins/`
3. Activez le plugin depuis l'écran des plugins dans WordPress
4. C'est tout ! Le bouton d'accessibilité apparaîtra automatiquement sur votre site

== Frequently Asked Questions ==

= Qui peut utiliser ce plugin ? =

Tous les types de sites WordPress qui souhaitent améliorer leur accessibilité.

= Les paramètres persistent-ils ? =

Oui ! Les paramètres sont sauvegardés soit dans un cookie (visiteurs non connectés) soit dans les métadonnées utilisateur (utilisateurs connectés).

= Puis-je personnaliser l'apparence du plugin ? =

Le plugin inclut des styles CSS que vous pouvez surcharger dans votre thème enfant.

= Est-ce que le plugin ralentit mon site ? =

Non. Le plugin est léger et optimisé pour ne pas impacter les performances.
